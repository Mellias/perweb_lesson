"Selamat Datang di Website Pertamaku"

Selamat datang di dunia html, html singkatan dari hyper text markup language yaitu bahasa dasar pemrograman untuk membuat website, bahasa ini sebagai landasan untuk belajar mendesain website.

Selain html ada juga bahasa pemrograman php dan javascript, php untuk membuat suatu website, akan tetapi kedua bahasa pemrograman tersebut ditulis di dalam bahasa pemrograman html.

Untuk itu html menjadi bahasa pemrograman wajib yang harus dikuasai untuk mendesain suatu website.
